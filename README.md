## 📌 Why this repo exists

I'm documenting my Git journey so I can:
- Track my progress over time
- Have a quick personal reference for commands I tend to forget
- Learn by writing things in my own words
- Look back and see how far I've come

## 🧠 Key concepts I want to remember

> - `git commit` saves a snapshot, not a diff.
> - `HEAD` is just a pointer to my current branch/commit.
> - `git --help` gives me the whole library.
